## 一、游戏概述

### 1.1 游戏简介

《边境守卫者》是一款奇幻题材的 2D 策略塔防游戏。玩家在固定路线的地图上，消耗金币建造并升级防御塔，阻止一波又一波的怪物突破防线。核心乐趣在于：**根据敌人特性与行进路线（含双入口与分叉），灵活搭配六种建筑（五战斗塔 + 矿场），并制定金币 / 钻石双轨升级策略。**

### 1.2 独特卖点（USP）

- **路线交互革新**：双出生点、汇合与分叉；部分敌人无视兵营或拆塔，迫使玩家调整布阵。
- **炮塔升级分支**：每种战斗塔五级成长，Lv.5 二选一终极分支（游戏内英文名如 Ranger / Siege）。
- **钻石经济**：Lv.4/Lv.5 消耗钻石；矿场产出 + 精英/BOSS 掉落 + 波次奖励。
- **敌人行为多样化**：治疗、自爆、召唤、无敌周期、拆塔、BOSS 两阶段等。
- **环境利用（计划中）**：古树缠绕、猎人陷阱等地图互动。

### 1.3 目标平台与操作

- **平台：** PC（鼠标）；后续可适配触屏。
- **操作（当前原型）：**
  - 底部栏选塔 → **先点击空闲高台**，再点塔标签（或 **1–6** 热键）建造；**拖拽**到高台仍可直接建造
  - 点击已建塔 → 左下信息面板升级 / 出售 / 选分支
  - 点击路径上的敌人 → 右下信息面板显示名称、HP、护甲/魔抗等级、威胁描述
  - 兵营 → **Rally** → 在号召范围内点击地图调动士兵
  - 右上角 **Call Early (+gold)** 提前开波并获得金币奖励
  - 备波阶段在上/下**出生入口**显示 **?** 情报按钮，点击查看该路线下一波敌人
  - 胜利后 **Continue** 返回开局主菜单（可调整难度/设置后再 Play）；**Reset**（右上角或评星面板）立即重开当前关卡
  - **Pause / Esc** 暂停或继续游戏；右上角 **Exit** 退出本局（确认后回主菜单）
  - 右上角 **1x / 2x / 3x** 切换战斗倍速（暂停或 Game Over 时不可用）
  - 底部栏 **1–6** 快捷键选塔；**已选高台**时按热键或点标签即建造；或**拖拽**塔图标到高台；Esc 取消拖拽
  - **悬浮高台**显示地形 Tooltip；**点击高台**仅选中（青色圈）用于建塔；选中后**悬浮底栏塔按钮**预览该塔射程圈
  - 顶部**波次进度条**显示当前波已出怪占该波总数百分比（Spawning 阶段实时更新）
  - 启动时 **Main Menu**（黑屏首页 → Play 加载条 → 地图；Codex / Settings 独立子页 + Back）
  - **Settings**：难度 Easy/Normal/Heroic、SFX 音量、UI 缩放、全屏、减少动效
  - 暂停 / Victory 面板显示各塔伤害、击杀与 **DPS** 统计
  - 精英/BOSS 出场时顶部横幅 + 大血条；炮塔受击时 **Tower under attack!** 提示 + 屏幕震动
  - 左上角 **Codex** 打开图鉴（**Towers / Enemies / Synergy / Map** 四页；自动暂停，Close 恢复）
- **界面语言：** 游戏内 UI 为**英文**；本 GDD 为中文设计说明。

### 1.4 核心循环

```
开始关卡 → 15 秒备波（可 Call Early）→ 双入口交替出怪 → 击杀获金/钻 →
升级塔（金 Lv.2–3，钻 Lv.4–5 分支）→ 应对精英/BOSS → 10 波全胜 → Victory → 评星获钥匙 → 解锁下一地图
```

## 二、核心玩法与机制

### 2.1 战斗规则

- **防守点：** 1 个终点 `(19,7)`；普通漏怪扣 1 生命，精英 5，BOSS 10；撕裂者**偷 10 金不扣生命**。
- **可建造位置：** 仅 **高台（BuildPlatform）**；格林姆森林共 **24** 个高台（其中 **12** 个为特殊地形格，颜色/标记区分）。
- **炮塔耐久（已实现）：** HP = **等级 × 50**（Lv.1 50 → Lv.5 250）；升级回满；自爆炎魔死亡爆炸、拆塔魔像攻击可摧毁炮塔（无出售退款）。
- **金币：** 起始 200；击杀掉落（见 4.2）；每波结束额外 `波次×8` 金。
- **钻石：** 矿场 0.1 钻/秒；精英 1–2 钻、BOSS 8 钻；第 4 波起每波结束 +1 钻。
- **波次：** 共 10 波；备波 **15 秒** 后自动开波；**Call Early** 获得 `14 + 波次×3` 金并立即开波（无钻石奖励）。
- **波次情报（已实现）：** 备波阶段在**上/下出生入口**显示 **?** 按钮；点击弹出该路线下一波敌人列表与战术提示（不再使用顶部 Incoming 面板）。
- **暂停（已实现）：** 点击 **Pause** 或按 **Esc**；暂停时冻结战斗、波次与备波倒计时；Game Over / Victory 时不可暂停并自动恢复时间流速。
- **战斗倍速（已实现）：** 右上角 **1x → 2x → 3x** 循环切换；暂停时不可改速；Game Over 重置为 1x。
- **战斗反馈（已实现）：** 伤害飘字；击杀 **+Ng / +ND** 飘字；敌人受击闪白 + 爆发圈；**SFX 策略**：普通命中无声、暴击/死亡/拾取/炮塔受击带冷却与主音量控制；拆塔等炮塔受伤时轻微震屏。
- **三星评价（已实现）：** 通关时按剩余生命评星；每获得**新**星级 +1 钥匙（累计最佳，可重复挑战刷星）。
- **钥匙解锁（已实现）：** 累计 **3 钥匙** 解锁地图二「熔岩裂谷」（场景尚未开发，胜利面板显示 Coming soon）。


| 星级  | 条件（20 生命起始） |
| --- | ----------- |
| ★   | 通关（生命 ≥ 1）  |
| ★★  | 剩余生命 ≥ 10   |
| ★★★ | 剩余生命 ≥ 18   |


- **HUD：** 左上角显示 `Keys: X · Best ★★☆`（当前关卡历史最佳）。
- **塔协同标签（已实现）：** 五战斗塔各有元素 Tag；**Lv.3 起**才有协同范围（蓝圈）并可触发/参与连携；一二级仅显示攻击/号召圈；选中塔时在信息面板显示 Tag 与协同状态（未满三级显示 `Synergy: unlocks at Lv.3`）；**Codex → Synergy** 分页提供总览、四组组合说明与布阵建议。

- **Lv.5 分支补全（已实现）：** Cannon Lv.2+ **眩晕**；Thunder **减护甲**；Destroyer **偷甲并 buff 邻塔**；Void Rift **普攻 + 虚空脉冲**。

### 2.2 炮塔建造与升级

- **建造：** 底部六按钮 → **先点空闲高台**（青色选中圈）→ 再点塔标签或 **1–6** 热键；选中高台后**悬浮**塔按钮可预览射程。
- **高台地形（P1.2）：** **悬浮**空闲高台显示 Tooltip（普通/特殊 + 效果）；**点击**仅选中建塔，不弹窗。
- **升级：** Lv.1→2→3 **金币**；Lv.3→4 **钻石**；Lv.4→5 **钻石 + 分支 A/B 二选一**。
- **出售：** 返还累计花费 **50% 金币**；钻石不退。
- **满级显示：** 信息面板标题与详情显示分支英文名（如 `Arrow (Ranger) Lv.5`）。

### P2 — 敌人与塔机制扩展

**目标：** 在 10 敌人 / 5 战斗塔框架上，增加「必须针对性应对」的单位与新 build 维度，强化已有协同 Tag 体系。

#### P2.1 新敌人类型（建议 4–6 种，分波次植入）

**目标：** 在不开发第二张地图的前提下，提供差异化玩法与重复游玩动力；将钥匙 / 星级系统从「解锁地图二」迁移为「解锁单图深度内容」。

| 模式 | 说明 |

**验收：** 至少「三选一词条」在 Normal 通关后解锁；与 `LevelProgressService` 记录最佳词条 build。
